# Employee Management System 🧑‍💼

A simple web application to manage employee records.

## 🔧 Technologies Used
- HTML
- CSS
- JavaScript

## ✨ Features
- Add new employee
- Edit existing employee details
- Delete employee from the list
- Basic form validation
- Stored in memory (no database)
